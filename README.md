# ReRead — secondhand book marketplace

## Features

- Уникальность: Площадка для покупки / продажи (обмена и поиска) книг и литературы
- Вариативность: Одна книга в разных оплетках и в разном состоянии за разные цены.
- Удобство: Поиск как БУ, так и новых книг с сайтов популярных магазинов
- Система блогов и рецензий: обсуждение книг, подборки для чтения, книжные коммьюнити

## Technologies

- Python - универсальный язык программирования с множеством библиотек
- Django Framework - полнофункциональный web-framework для языка Python
- Django Templates - инструмент для шаблонирования страниц
- HTML CSS JS - инструменты для разработки frontend
- Написание кода по БЭМ - методология web-разработки Блок, Элемент, Модификатор
- Адаптивная вёрстка, мобильная версия
- SQLite -  База Данных для данных любой сложности и размеров, совместимая с Django
- [Repl.it](http://Repl.it) - хостинг для размещения backend + frontend частей сервера
